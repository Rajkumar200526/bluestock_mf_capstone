Mutual Fund Analytics Platform
Project Overview

The Mutual Fund Analytics Platform is an end-to-end financial analytics solution developed during the Bluestock Fintech Capstone Project. The platform processes mutual fund datasets, performs data cleaning, calculates performance metrics, generates risk analytics, and provides interactive dashboards for investment analysis.

Project Objectives
Build ETL pipeline
Design SQLite database
Perform EDA
Calculate fund performance metrics
Develop Power BI dashboard
Implement risk analytics
